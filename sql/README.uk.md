# 🗄️ База Даних: Схеми та міграції (`sql/`)

Містить SQL скрипти для ініціалізації та початкового налаштування структури бази даних PostgreSQL.

## 📜 Порядок виконання

### 1. `01_create_schema.sql` (Data Definition Language)
Створює базову реляційну структуру системи.
- **Substations (`substations`)**: Головні вузли енергомережі.
- **Telemetry (`substation_data`)**: Сховище високочастотних часових рядів (Напруга, Струм, Потужність, Погодні умови).
- **Alerts & Logs**: Таблиці для моніторингу здоров'я системи.
- Реалізує гіпер-оптимізовані індекси для швидких OLAP-запитів по часових рядах.

### 2. `02_insert_static_data.sql` (Data Manipulation Language)
Заповнює базу даних початковими статичними налаштуваннями.
- Додає головні підстанції (наприклад, "ПС Київська-Центральна").
- Встановлює базові географічні та фізичні параметри, необхідні для симулятора `physics.py`.

---

## ⚙️ Автоматизована міграція
Вам не потрібно запускати ці скрипти вручну. Додаток має вбудовану систему самовідновлення бази даних. Під час запуску системи сервіс `src/services/data/migrate_db.py` автоматично виконає ці скрипти, якщо виявить, що база даних порожня.
