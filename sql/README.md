# 🗄️ `sql/` - База Даних

Містить SQL скрипти для ініціалізації та початкового налаштування PostgreSQL бази даних.

- `01_create_schema.sql` — Створення структури таблиць (телеметрія, підстанції).
- `02_insert_static_data.sql` — Заповнення довідників та початкових статичних даних.

Ці скрипти автоматично використовуються сервісом `src/services/data/migrate_db.py` під час розгортання проєкту.
