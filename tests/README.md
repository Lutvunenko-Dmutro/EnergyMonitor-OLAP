# 🧪 `tests/` - Тестування

Ця папка містить набір автоматизованих тестів, написаних за допомогою фреймворку `pytest`. Вони забезпечують стабільність критичних компонентів системи.

## Що тестується?

- `test_database.py` — Перевірка підключення та виконання транзакцій до БД.
- `test_core_analytics.py` — Валідація алгоритмів фільтрації та агрегації.
- `test_ml_model.py` — Завантаження ваг, скейлерів та базова валідація інференсу моделей прогнозування.
- `test_physics.py` — Юніт-тести для цифрового двійника (Digital Twin) та перевірки правильності розрахунку фізичних формул.
- `test_security.py` — Перевірка на відсутність SQL ін'єкцій та інших вразливостей.

Для запуску всіх тестів виконайте в консолі з кореня проєкту:
`pytest tests/`
