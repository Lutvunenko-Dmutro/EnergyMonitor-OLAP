# ЗАГАЛЬНІ ВИСНОВКИ (ЗВІТ З ПРАКТИКИ)

Під час проходження переддипломної практики на кафедрі інформаційних та комунікаційних технологій некомп'ютерних наук та інженерії програмного забезпечення мною було проведено комплексне дослідження за тематикою кваліфікаційної роботи.

**Основні результати практики:**
1.  **Опрацьовано теоретичну базу:** Вивчено концепції Smart City, Smart Grid та Digital Twin. Обґрунтовано доцільність застосування рекурентних нейронних мереж архітектури LSTM для предиктивного аналізу часових рядів енергоспоживання.
2.  **Проаналізовано існуючі рішення:** Проведено порівняння класичних статистичних методів (ARIMA) з сучасними підходами глибокого навчання, що дозволило виявити переваги LSTM у роботі з нелінійними та мультифакторними даними.
3.  **Розроблено проектні рішення:** Спроєктовано 4-рівневу архітектуру системи EnergyMonitor-OLAP, розроблено структуру бази даних за принципом «зірка» та реалізовано прототип інтелектуального ядра.
4.  **Проведено верифікацію:** На еталонних наборах даних підтверджено працездатність обраних алгоритмів. Оптимізована модель LSTM v3 демонструє точність (MAPE) в межах 1.5–3.1%, що відповідає промисловим стандартам.
5.  **Зібрано матеріал для диплома:** Матеріали звіту з практики повністю відповідають структурі майбутньої кваліфікаційної роботи та будуть використані при написанні її основних розділів.

Вважаю, що завдання переддипломної практики виконані у повному обсязі, а отримані результати є достатніми для подальшого завершення кваліфікаційної роботи бакалавра.
