# ВСТУП

Актуальність теми дослідження обумовлена дефіцитом генеруючих потужностей в енергосистемі України, що вимагає переходу від реактивного управління (реакція на аварію) до превентивного. Центральне місце в цій системі займає впровадження хмарної платформи, яка на основі потоків телеметрії формує прогноз на 48 годин наперед. Це дозволяє диспетчерським службам ідентифікувати ризики перевантаження вузлів до моменту спрацювання автоматики захисту.

**Об’єкт дослідження** — процес короткострокового прогнозування електричного навантаження та моніторингу технічного стану вузлів Smart Grid.

**Предмет дослідження** — методи глибокого навчання (LSTM) та OLAP-технології для аналізу часових рядів енергоспоживання.

**Мета роботи** — розробка та програмна реалізація хмарної аналітичної платформи EnergyMonitor-OLAP для моніторингу та предиктивної аналітики стану енергетичної інфраструктури.

Для досягнення мети вирішено наступні завдання:
1. Проаналізувати методи прогнозування енергоспоживання та обґрунтувати вибір архітектури LSTM.
2. Спроектувати чотирирівневу архітектуру системи з використанням хмарних OLAP-сховищ.
3. Розробити предиктивне ядро системи на основі нейронної мережі LSTM v3 з використанням тригонометричного кодування часових патернів.
4. Реалізувати інтерактивну візуалізацію даних на базі фреймворку Streamlit.
5. Провести тестування системи, оцінити точність моделей за допомогою метрики MAPE та розгорнути платформу через CI/CD конвеєр.

**Практичне значення отриманих результатів.** Розроблена платформа EnergyMonitor-OLAP дозволяє диспетчерським службам здійснювати планування на **48 годин наперед** за допомогою ковзного вікна прогнозування. Це дозволяє попередити каскадні відключення та перейти до обслуговування обладнання за його фактичним технічним станом.

**Структура та обсяг роботи.** Робота складається зі вступу, трьох розділів, загальних висновків, списку використаних джерел та додатків.
