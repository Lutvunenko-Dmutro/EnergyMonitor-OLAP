## 1.3. Постановка завдань практики

В рамках проходження практики необхідно вирішити такі завдання:
1.  **Проаналізувати логічну структуру** збору телеметричних даних у сучасних Smart Grid мережах.
2.  **Виконати порівняльний аналіз** класичних статистичних методів (ARIMA) та рекурентних нейронних мереж (LSTM) для прогнозування енергоспоживання.
3.  **Розробити архітектуру** цифрового двійника (Digital Twin) для симуляції навантаження підстанцій.
4.  **Зібрати набір навчальних даних** (датасет) на основі відкритих джерел та результатів роботи симулятора.
5.  **Верифікувати точність** розроблених моделей на тестових вибірках.
