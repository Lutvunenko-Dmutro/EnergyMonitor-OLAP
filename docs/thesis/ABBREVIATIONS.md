# ПЕРЕЛІК УМОВНИХ ПОЗНАЧЕНЬ ТА СКОРОЧЕНЬ

**AI (ШІ)** — Artificial Intelligence (Штучний інтелект)
**BI** — Business Intelligence (Системи бізнес-аналітики)
**CI/CD** — Continuous Integration / Continuous Deployment (Безперервна інтеграція та безперервне розгортання)
**LSTM** — Long Short-Term Memory (Довга короткострокова пам'ять — архітектура рекурентної нейронної мережі)
**MAE** — Mean Absolute Error (Середня абсолютна помилка)
**MAPE** — Mean Absolute Percentage Error (Середня абсолютна відсоткова помилка — метрика точності прогнозування)
**ML** — Machine Learning (Машинне навчання)
**OLAP** — Online Analytical Processing (Аналітична обробка даних у реальному часі)
**ONNX** — Open Neural Network Exchange (Відкритий формат обміну нейронними мережами)
**QA** — Quality Assurance (Методологія забезпечення якості програмного продукту)
**RMSE** — Root Mean Square Error (Середньоквадратична помилка)
**RNN** — Recurrent Neural Network (Рекурентна нейронна мережа)
**SaaS** — Software as a Service (Програмне забезпечення як сервіс)
