**ЗАСТОСУВАННЯ НЕЙРОМЕРЕЖ LSTM ТА КОНЦЕПЦІЇ ЦИФРОВИХ ДВІЙНИКІВ ДЛЯ МОНІТОРИНГУ ЕНЕРГОСПОЖИВАННЯ SMART CITY**

**Луценко Д. О.** (бакалавр), **Керівник: [Прізвище Керівника]** 
*Міжнародний науково-технічний університет імені академіка Юрія Бугая (МНТУ)*

**Анотація.** У роботі розглянуто підхід до створення інтелектуальної системи предиктивного моніторингу енергомереж. Запропоновано поєднання архітектури LSTM v3 з гармонійним кодуванням часових ознак та концепції цифрових двійників для розрахунку технічного стану обладнання. Результати тестування на еталонних даних підтвердили високу точність прогнозування (MAPE < 3.1%).

**Abstract.** This paper considers an approach to creating an intelligent system for predictive monitoring of energy grids. A combination of the LSTM v3 architecture with harmonic encoding of temporal features and the concept of digital twins for calculating the technical condition of equipment is proposed. Test results on reference data confirmed high forecasting accuracy (MAPE < 3.1%).

**Ключові слова:** Smart City, Smart Grid, LSTM, Цифровий двійник, Предиктивний аналіз.

---

**Вступ.** Сталий розвиток сучасної інфраструктури «Розумного міста» (Smart City) критично залежить від стабільності енергопостачання. Традиційні методи моніторингу зазвичай реагують на аварії постфактум, що призводить до значних економічних втрат. Перехід до предиктивного управління (Predictive Maintenance) вимагає впровадження високоточних алгоритмів прогнозування та створення віртуальних моделей фізичних об’єктів у реальному часі.

**Методи дослідження.** Для розв'язання задачі прогнозування часових рядів енергоспоживання було обрано рекурентну нейронну мережу архітектури LSTM (Long Short-Term Memory) версії v3. Ключовою особливістю реалізації є використання тригонометричної трансформації часових ознак ($x_{sin} = \sin(2\pi t / T), x_{cos} = \cos(2\pi t / T)$), що дозволяє нейронній мережі коректно сприймати циклічність добових та тижневих періодів. 

Інтеграція з концепцією **Цифрового двійника (Digital Twin)** реалізована через фізичний рушій, який розраховує непрямі показники надійності, зокрема **Health Score** та **H2 PPM** (вміст водню в маслі), базуючись на динаміці навантаження та температурі довкілля.

**Результати.** Програмна реалізація системи базується на 4-рівневій архітектурі (Streamlit UI, AI Core, PostgreSQL Data Layer, DevOps). Предиктивне ядро навчено на еталонному наборі даних PJM Dayton Hourly. Експериментально встановлено, що використання функції втрат **Huber Loss** робить модель стійкою до викидів. 

Основні метрики точності моделі LSTM v3:
*   Середня абсолютна похибка (MAPE): **1.5% — 3.1%**.
*   Коефіцієнт детермінації ($R^2$): **0.92**.
*   Обсяг верифікації: **79 автоматизованих тестів**.

**Висновки.** Створена інтелектуальна система **EnergyMonitor-OLAP** демонструє високу ефективність у задачах короткострокового прогнозування (24–48 годин). Поєднання методів глибокого навчання та імітаційного моделювання фізичних процесів забезпечує диспетчерські служби Smart Grid надійним інструментом для проактивного запобігання перевантаженням та аваріям.
