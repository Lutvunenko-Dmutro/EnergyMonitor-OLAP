# ВИСНОВКИ

У ході виконання кваліфікаційної роботи бакалавра було розв'язано актуальну науково-технічну задачу — розробку та впровадження інтелектуальної SaaS-платформи EnergyMonitor-OLAP для предиктивного аналізу та моніторингу енергоспоживання в інфраструктурі Smart City.

На основі проведених досліджень та розробок можна зробити такі висновки:

1.  **Аналітичне обґрунтування**: Аналіз сучасного стану енергосистем підтвердив критичну необхідність переходу від реактивного до проактивного управління (Predictive Maintenance). Встановлено, що використання рекурентних нейронних мереж є найбільш ефективним методом для обробки нелінійних та стохастичних часових рядів енергоспоживання.

2.  **Архітектурне рішення**: Спроєктовано та реалізовано багатошарову SaaS-архітектуру, що поєднує рівень вилучення даних, інтелектуальне ядро на базі LSTM v3 та інтерактивний аналітичний інтерфейс Streamlit. Використання концепції цифрового двійника (Digital Twin) дозволило моделювати не лише навантаження, а й фізичний стан обладнання (знос трансформаторів, втрати у ЛЕП), що значно підвищило інформативність системи.

3.  **Математичне забезпечення**: Розроблено та оптимізовано предиктивну модель на основі архітектури LSTM зі 128 юнітами вхідного шару та функцією втрат Huber Loss. Застосування тригонометричного кодування часу (sin/cos) дозволило усунути циклічні розриви в даних та покращити здатність моделі до виявлення добової сезонності.

4.  **Практичні результати**: Експериментальна перевірка системи на даних PJM Interconnection підтвердила високу точність прогнозування. Середня абсолютна похибка у відсотках (MAPE) становила 1.5% — 3.1%, що є зразковим показником для короткострокового прогнозування на 24–48 годин наперед. Автоматизація CI/CD процесів через Docker та Render забезпечила високу доступність та стабільність розгортання платформи.

5.  **Економічна стійкість та перспективи**: Впровадження EnergyMonitor-OLAP дозволяє знизити витрати на енергетичні небаланси та мінімізувати ризики аварійних відключень через перевантаження мережі. Система має високий потенціал для подальшого масштабування, зокрема для інтеграції з відновлюваними джерелами енергії та розумними системами обліку (Advanced Metering Infrastructure).

Мета роботи досягнута в повному обсязі, а розроблений програмний продукт готовий до практичної експлуатації в умовах цифровізації міської енергетичної інфраструктури.
