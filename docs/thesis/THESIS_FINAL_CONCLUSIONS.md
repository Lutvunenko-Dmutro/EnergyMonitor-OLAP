# ЗАГАЛЬНІ ВИСНОВКИ

У ході виконання кваліфікаційної роботи бакалавра проведено дослідження, проєктування та програмну реалізацію хмарної аналітичної платформи EnergyMonitor-OLAP. На основі отриманих результатів сформульовано наступні висновки:

1. Проведено аналіз методів прогнозування енергоспоживання, який показав обмеженість статистичних моделей ARIMA при роботі з нестаціонарними рядами. Обґрунтовано використання рекурентних мереж LSTM як інструменту для виявлення нелінійних залежностей у даних Smart Grid.
2. Розроблено та впроваджено чотирирівневу архітектуру системи на базі хмарного OLAP-сховища. Інтеграція концепції цифрових двійників у ядро системи дозволила в одному аналітичному запиті отримувати як прогноз навантаження, так і показник Health Score для кожного вузла мережі без залучення додаткових сервісів.
3. Програмно реалізовано інтелектуальний модуль на основі архітектури LSTM v3. Використання тригонометричного кодування часових ознак та функції втрат Huber Loss дозволило мінімізувати вплив аномальних викидів телеметрії та підвищити стабільність прогнозів.
4. Експериментальна перевірка системи на даних PJM Interconnection підтвердила високу точність розроблених моделей. Досягнутий показник середньої абсолютної похибки **MAPE = 3.08%** відповідає вимогам технічного завдання (< 4%).
5. Практичне значення роботи полягає у створенні SaaS-рішення, яке інтегрується у CI/CD процеси енергетичних компаній. Впровадження EnergyMonitor-OLAP дозволяє диспетчерським службам перейти до предиктивного обслуговування інфраструктури, що підвищує надійність енергозабезпечення міст в умовах дефіциту потужностей.
