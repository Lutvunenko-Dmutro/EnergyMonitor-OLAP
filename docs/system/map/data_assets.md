# 📊 Паспорт: Набори Даних (Data Assets)

> **Шлях:** `data/` та `data/fallback/`

## 📌 Призначення
Цей розділ описує набори даних (Datasets) та локальні бази даних, що використовуються в проєкті для симуляції, бектестування та навчання моделей машинного навчання.

## 📂 Файли
* **`.csv` файли:** Історичні часові ряди навантаження енергомереж (наприклад, AEP, COMED, DAYTON тощо).
* **`.db` файли:** Локальні файли баз даних SQLite, які використовуються для збереження кешу та симуляційних даних.
