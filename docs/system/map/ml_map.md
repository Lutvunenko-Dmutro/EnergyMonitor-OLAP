# 🧠 Мапа ШІ (ML Engine)

Цей розділ описує «мозок» системи — як ми готуємо дані, навчаємо LSTM та робимо прогнози в реальному часі.

## 📂 Підготовка Даних (`src/ml/vectorizer.py`)

**Навіщо:** Нейромережа не розуміє таблиці БД, вона розуміє «вікна» даних.
- **Як працює:** Перетворює часові ряди у 3D-тензори (Windows, Timesteps, Features).
- **Чому так:** LSTM потребує контексту (наприклад, останні 24 години), щоб побачити тренд.

---

## 📂 Прогнозування (`src/ml/predict_v2.py`)

**Навіщо:** Головний скрипт для генерації майбутнього навантаження.
- **Як працює:** Використовує рекурентне передбачення (бере прогноз на годину 1 і використовує його для прогнозу години 2).
- **Ключова фішка (Domain Adaptation):** Система автоматично підлаштовує масштаб прогнозу під потужність конкретної підстанції через `scale_factor`.

---

## 📂 Збереження Моделей (`src/ml/models/`)

**Навіщо:** Тут лежать «знання» системи.
- **`.h5 / .keras`**: Формати TensorFlow для навчання.
- **`.onnx`**: Високопродуктивний формат для використання на сайті.
- **`.pkl`**: Скейлери (запам'ятовують мінімуми та максимуми даних).
