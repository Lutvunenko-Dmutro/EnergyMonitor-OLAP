# 📚 `docs/` - Документація

Ця папка містить всю технічну та архітектурну документацію проєкту.

## ProperDocs System

Проєкт використовує власну систему паспортів (`system/map/`), де кожен Python файл має відповідний Markdown-паспорт з описом його призначення, залежностей та архітектури.

- Усі паспорти можна автоматично провалідувати, запустивши `scripts/audit_passports.py`.
- Папка також містить згенеровані HTML-версії інтерфейсу (`ui.html`, `ui/views/` тощо).

*(Примітка: Усі академічні/дипломні документи були переміщені в архівну папку, тому тут знаходиться лише технічна документація самого програмного продукту).*
