## 🔬 Наукова новизна та досягнення
Система базується на концепції **Digital Twin (Цифровий двійник)** і має наступні переваги:
1.  **Гібридна модель прогнозування:** Використання нейромереж LSTM (v3) разом із механізмом **SARIMA Fallback**, що забезпечує 100% доступність прогнозів при будь-якому стані мережі.
2.  **Recursive Time-Series Inference:** Алгоритм ітеративного передбачення на 24 години вперед через ONNX-конвеєр.
3.  **Фізична валідація:** Модель розрахунку теплової деградації трансформаторів та динамічних втрат потужності в AC/HVDC лініях (`src/core/physics.py`).
4.  **Zero-Flicker UI:** Оптимізація інтерфейсу Streamlit через фрагментарний рендеринг для оновлення KPI в реальному часі.

## 🚀 Як запустити демо (Developer Mode)
Якщо ви бажаєте перевірити систему в дії, виконайте наступні команди:

```bash
# 1. Клонування репозиторію
git clone https://github.com/Lutvunenko-Dmutro/EnergyMonitor-OLAP.git

# 2. Запуск через Docker (найшвидший шлях)
docker-compose up --build
```
*Система автоматично підключиться до хмарної бази даних Neon, завантажить скейлери та запустить інтерактивний дашборд.*
