# 🛠️ Утиліти, Моніторинг та Інструментарій (`scripts/`)

Папка містить різноманітні Python-скрипти, які не є частиною основного веб-додатка, але виконують важливі сервісні задачі: автоматизацію, бенчмаркінг моделей та аудит безпеки.

## 🌟 Ключові інструменти

### 🎛️ 1. Developer Dashboard (`system/dev_dashboard.py`)
Централізована панель управління для розробників. Замість того, щоб запускати скрипти через консоль, дашборд надає інтерактивний GUI для запуску аудитів, перевірки здоров'я БД та бенчмаркінгу ML-моделей.
*(Запускається через `START_DASHBOARD.py` у корені проєкту).*

### 🤖 2. Machine Learning Pipelines (`ml/`)
- **`benchmark_models.py`**: Запускає автоматизоване змагання між моделями прогнозування (ARIMA vs LSTM) на історичних даних, будує графіки точності та метрики R².
- **`audit_data.py`**: Глибокий аналізатор датасетів. Шукає аномалії, пропущені значення та перевіряє кореляцію ознак перед тренуванням моделі.
- **`real_data_evaluation.py`**: Перевіряє натреновані моделі на безперервному потоці симульованої/реальної телеметрії.

### 🛡️ 3. Аудит Архітектури (`system/`)
- **`diagnose.py`**: Проводить глибокий аудит коду на предмет вразливостей безпеки (SQL Injection), витоків пам'яті та архітектурних помилок. Генерує інтерактивний HTML-звіт.
- **`check_db_stats.py`**: Моніторить здоров'я PostgreSQL, знаходить найбільші таблиці та відслідковує продуктивність запитів.

### 📄 4. Управління Документацією
- **`audit_passports.py`**: Перевіряє цілісність усіх архітектурних Markdown-документів (ProperDocs).
- **`rebuild_master_index.py`**: Автоматично оновлює навігаційні посилання у системі документації.
