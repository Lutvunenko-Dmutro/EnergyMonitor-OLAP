# 🧠 `src/` - Ядро системи (Source Code)

Ця папка містить весь основний вихідний код проєкту. Архітектура побудована за модульним принципом, що дозволяє легко масштабувати систему.

## Структура директорії

- `app/` — Головна конфігурація додатку та загальні типи даних.
- `core/` — Критичні компоненти:
  - `analytics/` — Базова агрегація, кластеризація (K-Means) та фільтрація даних.
  - `database/` — Data Access Layer (DAL) для роботи з PostgreSQL через SQLAlchemy.
  - `diagnostics/` — Система аудиту безпеки та здоров'я коду.
  - `physics.py` — Фізичний рушій (Digital Twin), що розраховує втрати потужності та вплив погоди.
- `ml/` — Machine Learning Pipeline:
  - Код тренування моделей (LSTM, ARIMA).
  - Механізм перевірки точності та бенчмаркінгу (Metrics Engine).
  - Скомліовані ваги моделей (`models/`).
- `services/` — Бізнес-логіка, Data-mining та симулятор телеметрії (`data_generator.py`).
- `ui/` — Компоненти користувацького інтерфейсу (Streamlit):
  - Поділено на `components` (перевикористовувані віджети), `segments` (блоки) та `views` (повноцінні екрани).
- `utils/` — Допоміжні функції, валідатори та налаштування логування.
